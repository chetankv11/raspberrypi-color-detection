# Raspberry Pi Color Detection using Pi Camera

This project detects colored objects using a Raspberry Pi camera and OpenCV.

## Features
- Real-time color detection
- HSV color segmentation
- Bounding box detection
- Works with Raspberry Pi Camera (PiCam)

## Hardware Required
- Raspberry Pi
- Raspberry Pi Camera Module
- Python 3

## Installation

Install dependencies:

pip install -r requirements.txt

## Run the Program

python color_detection.py

## Applications
- Color tracking robot
- Object detection
- Robotics and automation
